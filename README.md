# Social-text-nlp-analysis-in-China
Analysis of social media text using sentiment classification and keyword extraction.
This project analyzes Chinese social text data using NLP techniques.

## Tasks
- Sentiment analysis using Tencent Cloud NLP API
- Keyword extraction using Jieba

## Data
Two independent datasets:
- Elderly-related text data
- News/social media text data

## Output
- Sentiment results
- Word frequency analysis

## Tools
- Python
- Jieba
- Tencent Cloud NLP API
